Uri Alias and Hebrew Date Service

External Uri: https://urialiasservice.azurewebsites.net/

Uri and Alias Service

Request

POST /createUri

Values in a json format

                        
                            {
                                "AKNType": String,Example: act
                                "Type": String, Example: law,
                                "emissionDate": String,Must to be as YYYY-MM-DD format, Example:"2021-07-11",
                                "DocumentNumber": String, Example: "123456"
                                "Language": String, Example:"heb",
                                "ExpressionDate": String,
                            }

Return

Array with Uri at 0 index and Alias at 1 index

                        
                            [
                                "/akn/il/act/law/2021-07-11/123456/heb@",
                                "/akn/il/act/‏חוק‎/‏ב-אב-התשפא‎/123456/heb@"
                            ]

Date Service

Gregorian to Hebrew Date Service

Request

POST /gre2hebDate

Values in a json format

                            
                                {
                                    "year": "2021" ,
                                    "month": "07",
                                    "day": "11"
                                }

Return

Hebrew Date as YYYY-MM-DD format

                            
                                    ‏ב-אב-התשפא‎

Request

GET /gre2hebDate/:date

Get date YYYY-MM-DD Gregorian format in the uri param

                            
                                /gre2hebDate/:date
                                :date => YYYY-MM-DD =>2021-07-11
                                Example: /gre2hebDate/2021-07-11

Return

Hebrew Date as YYYY-MM-DD format

                            
                                    ‏ב-אב-התשפא‎

Hebrew to Gregorian Date Service

Request

GET /heb2greDate/:hDate

Get date YYYY-MM-DD Hebrew format in the uri param

                            
                                /gre2hebDate/:date
                                :date => YYYY-MM-DD =>‏ב-אב-התשפא‎
                                Example: /gre2hebDate/‏ב-אב-התשפא

Return

Gregorian Date as YYYY-MM-DD format

                            
                                    ‏2021-07-12
